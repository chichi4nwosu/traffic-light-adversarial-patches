# Sensor Disagreement Detection Project

## Research Question

Can sensor disagreement metrics be used to detect faulty or spoofed sensor measurements in autonomous drones?

## Goal

Develop a lightweight sensor fusion safety monitor that detects abnormal sensor behavior.

## Sensors

- GPS
- Camera
- LiDAR
- IMU

## Simulation Platform

- CARLA

- Conference Paper
