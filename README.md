Adversarial Perception Security

University of Alabama REU 2026

Overview

This repository contains research, experiments, and notes related to adversarial attacks and perception security in autonomous systems.

The primary focus is understanding how machine learning-based perception systems can be manipulated through adversarial examples and adversarial patches, with an emphasis on autonomous vehicle traffic light detection.

Current work is centered around reproducing and extending results from:

Fool the Stoplight: Realistic Adversarial Patch Attacks on Traffic Light Detectors
Adversarial Patch
Related autonomous systems security and perception papers
